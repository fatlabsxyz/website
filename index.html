what we do?

Fat Solutions is a crypto dev shop that maxes out on zero-knowledge but doesn’t stop there. We’re focused on using new zk primitives to unlock fresh potential on the application layer — think private social logins, identity tools, wallets, or private token standards that actually change the game. We’re tapping into what zk can do to build things that deliver real value to users, not just hype.

mission?

We’re a team of engineers and researchers deep into zk tech, shaping it into apps that prioritize privacy and function, proving zero-knowledge can deliver in the real world. It’s still early for zk, and we’re here to build practical solutions that show its power now while setting the stage for what’s ahead.

0xBow - Privacy Pools

A privacy protocol for private, non-lazarus asset transfers on ETH (and L2s). Building and maintaining the SDK and Relayer components of the protocol.

# links: github repo

Starknet - Sumo Login

A zero-knowledge social login system using JWT proofs and account abstraction. Sumo lets users access smart wallets with Web2 identities like Google or Discord, without seed phrases or browser extensions. It generates zkProofs from signed JWTs and client-side session keys, enabling secure, seamless authentication and transaction signing on Starknet.

# links: github repo - docs

Miden - Wallet Research

Researching ways to integrate existing applications/wallets to onboard new users to the Miden private ecosystem.

Aztec - Terry Escape

Our friend Terry has to escape (or kill) some agents in order to win his psychological war. A multiplayer private shared-state game using oblivious transfers, homomorphic encryption and zero-knowledge proofs, built with Noir.
