What we do:

Fat Solutions is a crypto dev shop that maxes out on zero-knowledge but doesn’t stop there. We’re focused on using new zk primitives to unlock fresh potential on the application layer — think private social logins, identity tools, wallets, or private token standards that actually change the game. We’re tapping into what zk can do to build things that deliver real value to users, not just hype.

Our Mission:

We’re a team of engineers and researchers deep into zk tech, shaping it into apps that prioritize privacy and function, proving zero-knowledge can deliver in the real world. It’s still early for zk, and we’re here to build practical solutions that show its power now while setting the stage for what’s ahead.


Beyond that, we’re about pushing crypto forward. From zk’s bleeding edge to broader blockchain challenges like optimizing gas costs or refining UX for decentralized systems, our mission is to turn messy problems into clean, working code. ZK’s where we shine brightest, but we’re ready to tackle anything in the stack.

Partners and Projects:


Starknet - Sumo Login: A zero-knowledge social login system using JWT proofs and account abstraction. Sumo lets users access smart wallets with Web2 identities like Google or Discord, without seed phrases or browser extensions. It generates zkProofs from signed JWTs and client-side session keys, enabling secure, seamless authentication and transaction signing on Starknet.

# links: github repo - docs


Aztec - Terry Escape: Our friend Terry has to escape (or kill) some agents in order to win his psychological war. A multiplayer private shared-state game using oblivious transfers, homomorphic encryption and zero-knowledge proofs, built with Noir.

# links: github repo


0xBow - Privacy Pools: A blockchain protocol for private, non-lazarus asset transfers. We are building the SDK and Relayer for seamless integration and private withdrawal support.
